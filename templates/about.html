About AutoML Prediction System

The AutoML Prediction System is an intelligent web-based platform that automatically trains and deploys Machine Learning models without requiring manual coding or ML expertise. Users can upload their datasets, and the system will automatically perform data preprocessing, model training, evaluation, and deployment.

The system supports both Classification and Regression problems. It automatically detects the dataset type, selects the best-performing model, and deploys it for real-time predictions through a user-friendly interface.

Key Features:

  • Upload any CSV dataset for automatic training
  • Automatic data preprocessing and feature encoding
  • Automatic model selection and training
  • Supports Classification and Regression
  • Automatic deployment of trained model
  • Dynamic prediction interface based on dataset features
  • MLflow integration for experiment tracking
  • FastAPI-based high-performance backend

This system demonstrates a complete end-to-end AutoML pipeline, making Machine Learning accessible, scalable, and easy to use for real-world applications.
